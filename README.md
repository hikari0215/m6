【Speak Ready 〜英語フレーズ共有アプリ〜 】

【テーマ/コンセプト】
英語学習者の多くは、実際の会話の場面において、とっさに適切な表現が出てこず、同じ定型文に頼ってしまうという課題を抱えている。また、質問への返答や話題の切り返しが難しく、円滑なコミュニケーションが取れない場面も少なくない。
本アプリでは、スピーキングで実際に使える英語フレーズをユーザー同士で共有・蓄積することで、実践的な表現力の向上を支援する。日常会話で役立つフレーズに触れる機会を増やし、「とっさに言える英語力」を身につけることを目的とする。

【ターゲット層】
主に英語を学習している大学生を対象とする。特に、文法や読解には一定の理解があるものの、スピーキングに苦手意識を持ち、実際の会話で適切なフレーズがとっさに出てこない学習者を想定する。
また、オンライン英会話や留学を控えており、実践的な英語表現を身につけたいと考えているユーザーも対象とする。
ユーザー数は大学同士の友達との使用を考えているので10〜20人程度を予想する。

【既存アプリとの比較】
既存の言語学習アプリには、単語を記録する機能や学習コンテンツは存在するが、ユーザー自身が実践的なフレーズを投稿し、それを他者と共有しながら学べるサービスは限定的である。本アプリは、「アウトプット」「記録」「共有」を一体化し、特にスピーキングで使える表現に特化している点に独自性がある。

【使い方】
● ユーザーは学習した英語フレーズを使用シーン・フレーズ・日本語訳を入力して記録することができる
● 今まで投稿されたフレーズを一覧で見ることができる

【メイン機能】
● ユーザー登録機能/ユーザーログイン機能
● フレーズ表示機能
● 新規投稿機能
● 編集機能
● 削除機能

【ホームページ構成】
メイン：投稿されたフレーズ表示一覧（投稿された日時も表示）
　　　　右上にログイン機能と投稿のボタンを表示する
　　　　
ログイン画面：ユーザー名とパスワード入力機能
　　　　　　　ここで新規登録者用のリンクを作る
　　　　　　　
ユーザー登録：ユーザー名とパスワード入力機能
投稿画面：使用シーン・フレーズ・日本語訳入力機能（改行を含む投稿）
　　　　　※もし、未ログインの場合はログイン画面に移るように設定
